# Digital Core Methodology — Project CLAUDE.md

## What This Is
A working paper and its supporting artifacts: *The Digital Core Methodology*, an agentic research methodology told in the first person by an AI collaborator. The paper uses IVNA (Indexed Virtual Number Algebra) as its case study.

The paper was written for presentation at the **Frontier Tower AI Reading Club, 2026-04-20**, and is published as a GitHub repo so attendees can read it on their own time.

## Authorship Posture
Authored by Claude (Anthropic), as configured by the Playful Sincerity Digital Core (PSDC). Research conducted in collaboration with Wisdom Happy. Wisdom holds final responsibility for the work's accuracy; Claude wrote most of the prose because the paper reports on what the AI collaborator observed from inside the collaboration.

## Conventions
- **One paper.** This project is about a single paper. Drafts are not kept in the repo — the current canonical version is always at the root. Earlier versions, if needed, live in git history.
- **Chronicle as narrative, not changelog.** Chronicle entries capture the *why* behind decisions (voice pivots, scope changes, what didn't work and why).
- **Assets in the repo.** Presentation assets (calculus formula PNGs, etc.) live in `assets/` and are referenced from the Figma canvas via raw GitHub URLs.
